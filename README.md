# LM_IJKMoviePlayer

### 该项目基于b站的开源项目ijkplayer 在此基础上增加了一些基础功能 <br /> 

1.左侧屏幕上下滑动调整亮度
2.右侧屏幕上下滑动调整音量
3.横向左右拖动改变播放进度
4.横竖屏切换

### 支持播放格式 <br /> 

1.支持rtmp直播拉流
2.视频云播


由于ijk的framework比较大就没有放入项目内上传，请自行点击下方链接自取，解压缩后放入项目内即可
